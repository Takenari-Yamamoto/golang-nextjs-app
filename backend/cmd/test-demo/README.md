Go 言語のテストファイルを一括で実行するには、以下のコマンドを使います。

```
go test ./... -v
```

このコマンドは、カレントディレクトリおよびすべてのサブディレクトリ内のテストファイルを再帰的に検索し、見つかったすべてのテストを実行します。-v フラグは、テストの詳細な出力を表示するために使用されます。詳細な出力が不要な場合は、-v フラグを省略できます。

```
go test ./...
```

また、特定のディレクトリ内のすべてのテストファイルを実行するには、次のコマンドを使います。

```
go test path/to/directory/... -v
```

これは、指定されたディレクトリとそのサブディレクトリ内のテストファイルを再帰的に検索し、見つかったすべてのテストを実行します。

これらのコマンドを使用して、プロジェクト内の複数のテストファイルを一括で実行できます。
